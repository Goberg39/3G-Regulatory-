Getting started with this blog

This post explains how the site is put together so you can start publishing right away. When you're done reading, delete this file.

How a post gets published

There are two steps:

  1. Duplicate one of the files in the posts/ folder and rename it — for example posts/my-first-post.html. Edit the title, date, and body inside.
  2. Open index.html and copy one of the list items in the post list, pointing it at your new file. Newest posts go on top.

Where things live

  • index.html — the home page and list of posts
  • about.html — the about page
  • posts/ — one HTML file per post
  • css/style.css — all styling and theme colors
  • js/main.js — small touches like the footer year

Changing the look

Open css/style.css. The block at the very top holds the colors and fonts as variables — change --accent to recolor every link and highlight at once, or swap the fonts in --font-display and --font-ui. Dark mode is already wired up and follows the reader's system setting.

That's everything. Happy writing.
